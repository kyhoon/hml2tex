# hml2tex
Try to convert from .hml to .tex

### Usage
```bash
python main.py -i INPUT.txt -o OUTPUT.txt -c CONVERT_MAP.pkl
```
* INPUT : the .hml file to convert, saved in .txt format
* OUTPUT : the .tex file converted, saved in .txt format
* CONVERT_MAP : the .pkl file that contains the convert map between script symbols in Hangul and Latex
